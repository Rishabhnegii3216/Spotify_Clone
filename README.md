# Spotify_Clone
Built a Spotify-inspired web music player using HTML, CSS, and JavaScript. Implemented dynamic album and playlist loading, Audio API controls (play, pause, seek, volume), and responsive sidebar navigation. Strengthened skills in DOM manipulation, async/await, Fetch API, and interactive UI design with real-time media handling.


🎵 Spotify Clone – Web Music Player

A Spotify-inspired web music player built using HTML, CSS, and JavaScript.
This project replicates core Spotify UI features including dynamic playlists, album cards, responsive sidebar navigation, and real-time music playback controls.

🚀 Features

🎶 Dynamic album rendering from folder structure

📂 Playlist loading from local song folders

▶️ Play / Pause functionality

⏮ Previous / Next track controls

⏱ Real-time song timer & seekbar

🎚 Volume control & mute toggle

📱 Responsive sliding sidebar

🎨 Clean UI with hover effects
